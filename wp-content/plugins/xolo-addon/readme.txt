== Description ==

Xolo Addon gives you attractive Elementor widget to your websites. Its perfect test for <a href="https://xolotheme.com/themes/" rel="nofollow">Xolo Theme</a>, But You can use for another theme also Astra, Sinatra and many more.

**Available Widgets:**

&#9989; Slider Widget
&#9989; Features Widget
&#9989; About Widget
&#9989; CEO Box Widget
&#9989; Funfact Widget
&#9989; Info Widget
&#9989; Service Widget
&#9989; Section Title Widget
